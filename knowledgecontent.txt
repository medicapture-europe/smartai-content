=== MVR API DOCUMENTATION ===

BASE URL: http://{device-ip}:3333/api/
(Enable in Advanced Settings > Connections > Remote Access)

AUTHENTICATION: HTTP Basic Auth with MD5-encrypted password
SESSION MANAGEMENT: POST /api/begin returns X-session header; 5-min timeout

KEY ENDPOINTS:
- GET /api/device - Device info (no auth)
- GET /api/cameras - Connected cameras
- PUT /api/study/start - Start study with patient metadata
- PUT /api/study/snapshot - Capture image
- PUT /api/study/record/start|stop|pause|resume - Recording control
- GET /api/stream/video?pin=<pin> - Video stream (fMP4)
- GET /api/notify - Real-time notifications (chunked JSON)
- POST /api/pacs - Schedule DICOM upload
- GET /api/worklist - Patient list from worklist server
- PUT /api/system/reboot - Restart device
- GET /api/settings - Configuration values
- POST /api/study/overlays - Create custom overlay

RS232 PROTOCOL: Single-line format: <method> <path> [id] [data>\n

ERROR CODES: 2xx=Success, 400=Invalid params, 401=Unauthorized, ≥300=Failure
